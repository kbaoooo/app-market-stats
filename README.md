# App Store vs Google Play — Keyword & Category Data Report 2025

A comprehensive, single-page data report comparing the Apple App Store and Google Play Store. Built as an open resource for indie developers and App Store Optimization (ASO) practitioners.

## 📊 Overview
This project is a static HTML/CSS/JS dashboard that compiles and visualizes data from 7,500+ top apps across every store category. It highlights what users actually search for, which categories are exploding, and where the real opportunities lie for indie developers in 2025/2026.

## ✨ Features
- **Platform Comparison**: High-level metrics comparing downloads, revenue, and user behavior between iOS and Android.
- **Keyword Insights**: Breakdown of top branded and non-branded keywords.
- **Category Breakdown**: Deep dive into downloads, revenue, and dominant keywords by category.
- **Opportunity Analysis**: Actionable insights for indie developers based on current trends, competition levels, and monetization strategies.
- **Interactive UI**: Simple, fast, and responsive tabbed interface built with vanilla HTML, CSS, and JavaScript.

## 🚀 How to Use
This is a purely static project with no build steps or dependencies.

1. Clone the repository or download the files.
2. Open `index.html` in your favorite web browser.
